# Icon Contract Seed

Icons are visual semantics, not decoration.

## Delete

- Icon: `Trash2`
- Intent: `delete` / `destructive` / `danger`
- Must use shared action style source.

## Edit

- Icon: `Pencil`
- Intent: `edit`
- Placement: entity header/action cluster.

## Save

- Icon: `Save` where icon is useful.
- Intent: `save`.
- Placement: form footer/edit panel footer.

## Add

- Icon: `Plus` where icon is useful.
- Intent depends on action: add-note, add-task, add-case, add-event.

## Copy

- Icon: `Copy`.
- Placement: inline info row action.
