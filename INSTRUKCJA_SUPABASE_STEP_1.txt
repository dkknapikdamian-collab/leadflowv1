ETAP 1 — CO MASZ ZROBIĆ W SUPABASE, KROK PO KROKU

1. Wejdź do swojego projektu Supabase.
2. W lewym menu kliknij: SQL Editor.
3. Kliknij: New query.
4. Otwórz plik:
   SUPABASE_SQL_STEP_1.sql
5. Skopiuj CAŁĄ zawartość tego pliku.
6. Wklej do nowego query w Supabase.
7. Kliknij Run.

CO MA SIĘ STAĆ
- query ma zakończyć się bez czerwonego błędu,
- na końcu ma zwrócić małą tabelkę z:
  status = ok

JEŚLI WYSKOCZY BŁĄD
- nie poprawiaj nic sam,
- skopiuj cały czerwony komunikat i wklej mi go 1:1.

PO SQL
1. Wgraj overlay do repo.
2. Zrób:
   npm.cmd install
   npm.cmd run lint
   npm.cmd run build
3. Dopiero potem push.

CO TEN PAKIET ROBI

- przepina delete sprawy na Supabase,
