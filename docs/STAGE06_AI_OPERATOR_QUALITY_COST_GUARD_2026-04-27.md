# Stage 06 — AI Operator: jakość i kontrola limitów

Cel: asystent ma działać bardziej jak operator aplikacji, a nie jak zwykły chatbot.

Zakres:
- typy frontu znają pełny snapshot aplikacji,
- odpowiedzi z danych aplikacji nie zużywają dziennego limitu AI,
- pytania o szkice AI są obsługiwane regułowo z danych aplikacji,
- dalej obowiązuje zasada: brak komendy zapisu = czytanie, komenda zapisu = szkic.
